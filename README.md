# EV Charging Station Dynamic Scheduling System

基于 HAPPO-GNN-RL 的电动汽车充电站动态调度优化系统

## 功能

- 实时充电调度决策
- 多算法对比（MH-Res-GAT vs HAPPO-GNN-RL）
- 支持任意城市充电网数据导入
- 模型训练与评估

## 技术栈

- 后端: Flask + PyTorch
- 前端: HTML/CSS/JavaScript + Chart.js
- 数据库: SQLite

## 快速开始

```bash
cd webapp
pip install -r requirements.txt
python app.py
```

访问 http://localhost:5000
